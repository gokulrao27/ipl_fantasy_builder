1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## SEO verification files

The repo includes static files in `public/` for Google indexing setup:

- `googleeec3e9d38b44b337.html` (Google Search Console verification file)
- `sitemap.xml`
- `robots.txt`

Before production deployment, replace `https://yourdomain.com` in:

- `public/sitemap.xml`
- `public/robots.txt`

with your real domain.
